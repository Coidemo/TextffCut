# 実動作テスト計画書

## テスト環境

### 必要なもの
1. **テスト用動画ファイル**

   - 長い動画（30分以上）: 負荷テスト用

2. **API設定**
   - OpenAI APIキー（有効なもの）

3. **テスト環境**
   - Python環境（requirements.txtのインストール済み）

   - 十分なディスク容量

### 1. APIモードの正常系テスト

#### 1.1 基本的な文字起こし＋アライメント

## デバッグ用コード

### ログ出力強化
```python
# main.pyの適切な場所に追加
import logging
logging.basicConfig(level=logging.DEBUG)
logger = logging.getLogger(__name__)

# wordsフィールドの状態を詳細に出力
def debug_words_status(result):
    if hasattr(result, 'segments'):
        total_segments = len(result.segments)
        segments_with_words = sum(1 for seg in result.segments 
                                 if hasattr(seg, 'words') and seg.words)
        logger.info(f"Words状態: {segments_with_words}/{total_segments} セグメント")
        
        # 最初の数セグメントの詳細
        for i, seg in enumerate(result.segments[:3]):
            if hasattr(seg, 'words') and seg.words:
                logger.info(f"  セグメント{i}: {len(seg.words)}words")
            else:
                logger.warning(f"  セグメント{i}: wordsなし!")
```
