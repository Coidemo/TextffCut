# 🎙️ Buzz Clip

動画の文字起こしと切り抜きを効率化するツール

## 概要

Buzz Clipは、YouTube等の動画から高精度な文字起こしを行い、必要な部分だけを切り抜いて編集ソフトで使えるファイルとして出力するツールです。

### 主な機能

- **高精度な文字起こし**: WhisperXを使用した日本語対応の文字起こし
- **直感的な切り抜き編集**: テキストベースで切り抜き箇所を選択
- **無音削除**: 切り抜いた動画から無音部分を自動削除
- **多様な出力形式**: 
  - 動画ファイル（MP4）
  - FCPXMLファイル（Final Cut Pro用）
  - SRT字幕ファイル
- **設定の永続化**: よく使う設定を自動保存

## インストール

### 必要な環境

- Python 3.8以上
- FFmpeg（動画処理用）

### セットアップ

```bash
# リポジトリのクローン
git clone https://github.com/yourusername/buzz-clip.git
cd buzz-clip

# 依存関係のインストール
pip install -r requirements.txt
```

## 使い方

### 起動

```bash
streamlit run main.py
```

### 基本的な使用手順

1. **動画ファイルの選択**
   - 動画ファイルのフルパスを入力

2. **文字起こし**
   - Whisperモデルを選択（large-v3推奨）
   - 「新しく文字起こしを実行」をクリック

3. **切り抜き箇所の指定**
   - 左側の文字起こし結果から必要な部分をコピー
   - 右側のテキストエリアにペースト
   - 「更新」ボタンで確定

4. **処理オプションの選択**
   - 処理タイプ：「切り抜きのみ」または「無音削除付き」
   - 出力形式：「動画ファイル」または「FCPXMLファイル」
   - 字幕生成：必要に応じてSRT字幕を生成

5. **処理実行**
   - 「処理を実行」ボタンをクリック
   - 出力ファイルは指定したディレクトリに保存

