# 文字起こし修正ツール

`transcribe_for_lora.py` で生成した JSON を、ブラウザ上で音声を聞きながら修正するための
ローカル Web ツール。フィラー取りこぼしの手動補強を効率化する目的で作成。

## 起動

```bash
cd experiments/whisper_lora/edit_tool

python3 server.py \
  ../data/raw/20260129_生成AIの世界で情報収集が完結する人が増えている.json \
  /Users/naoki/myProject/TextffCut/videos/20260129_生成AIの世界で情報収集が完結する人が増えている.mp4
```

自動でブラウザが開きます（http://localhost:8000/）。
ブラウザを開かせたくない場合は `--no-browser` オプション。

## 使い方

1. 左の **▶ ボタン**でそのセグメントの音声を再生
2. テキスト欄でフィラー追記・修正
3. BGM 区間など学習から除外したいものは **Skip** にチェック
4. **⌘S** または「保存」ボタンで保存（30 秒毎に自動保存もあり）

すべてのセグメントを上から順に確認する想定。Skip 以外はすべて学習データ化されるため、
「人間が全部目を通した」前提で扱われる。

## キーボードショートカット

| キー | 動作 |
|------|------|
| ⌘S / Ctrl+S | 保存 |
| スペース | 選択中セグメントを再生（テキスト編集中を除く） |
| Esc | 再生停止 |

## 出力ファイル

保存ボタンを押すと、以下のファイルが生成/上書きされます:

```
{元 JSON}.edited.json
例: data/raw/20260129_....edited.json
```

構造:
```json
{
  "language": "ja",
  "model": "mlx-community/whisper-large-v3-mlx",
  "speed": 1.0,
  "source": "/path/to/video.mp4",
  "edited_at": "2026-04-21T...",
  "segments": [
    {
      "start": 38.4,
      "end": 56.3,
      "text": "あのー、最初の方に、えー、本編をちょっと...",   // 修正後
      "original_text": "あのー、最初の方に本編をちょっと...",   // 修正前
      "skip": false,
      "words": [...],
      "chars": [...]
    }
  ]
}
```

- `text`: 修正後（= 学習の教師信号）
- `original_text`: Whisper が出力した元テキスト
- `skip`: 学習データから除外するか（true なら使わない）

## 色の凡例

- **白**: 未変更
- **オレンジ枠**: テキストに変更あり
- **赤**: Skip マーク（学習から除外）
- **青枠**: 現在選択中のセグメント

## キャッシュ

初回起動時、動画から MP3 を抽出して `edit_tool/.audio_cache/` に保存します
（以降は再利用）。ディスク容量節約には手動削除してください。

## 依存

- Python 3.11+ (標準ライブラリのみ)
- ffmpeg (音声抽出用。TextffCut で既にインストール済みのはず)
