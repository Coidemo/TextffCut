# Adapters Layer (アダプター層)

内側の層（ドメイン、ユースケース）と外側の層（UI、外部サービス）をつなぐ変換層です。

## 特徴
- データ形式の変換を担当
- 依存性逆転の原則を実現
- インターフェースの実装を提供

## ディレクトリ構造

### controllers/
UIからの入力を受け取り、ユースケースを実行
- 入力データの検証
- ユースケースの呼び出し
- エラーハンドリング

### presenters/
ユースケースの結果をUIに適した形に変換
- ドメインオブジェクトをDTO（Data Transfer Object）に変換
- UI表示用のフォーマット
- エラーメッセージの整形

### gateways/
外部サービスとの接続を抽象化
- ユースケースが必要とするインターフェースの実装
- 外部APIやサービスとの通信をラップ
- データの変換とエラーハンドリング
