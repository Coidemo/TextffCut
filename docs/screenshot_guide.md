## 動画GIFで撮影するもの

1. **文字起こし実行の流れ**（30秒程度）
   - 動画選択 → 設定 → 実行 → 完了

2. **テキスト編集の流れ**（30秒程度）
   - コピー → ペースト → 更新 → ハイライト確認

3. **処理実行の流れ**（20秒程度）
   - オプション選択 → 実行 → 完了

## 撮影時の注意点

1. **個人情報の削除**
   - ファイル名に個人名が含まれていないか確認
   - パスに個人情報が含まれていないか確認

2. **見やすさ**
   - ブラウザは適度なサイズに（1280x800程度）
   - 不要なタブは閉じる
   - デスクトップは整理しておく

3. **サンプル動画**
   - 著作権フリーの動画を使用
   - または自作のテスト動画を使用

## 推奨ツール

- **スクリーンショット**: macOS標準（Cmd+Shift+4）
- **動画GIF作成**: 
  - Gifox（Mac）
  - ScreenToGif（Windows）
  - または QuickTime → オンライン変換
