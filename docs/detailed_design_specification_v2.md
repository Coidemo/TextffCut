## 2. システム構成

### 2.1 全体アーキテクチャ

システムは5つの層で構成されます：

| レイヤー | 役割 | 主要コンポーネント |
|---------|------|------------------|
| **プレゼンテーション層** | ユーザーインターフェース | Streamlit UI |
| **アプリケーション層** | 業務フロー制御 | ワークフローコントローラー、セッション管理 |
| **ビジネスロジック層** | 中核処理 | 文字起こし、動画処理、エクスポート |
| **データアクセス層** | データ永続化 | キャッシュ、設定管理 |
| **インフラ層** | 外部ツール連携 | FFmpeg、WhisperX、YouTube-DL |

### 2.2 モジュール構成

```
TextffCut/
├── main.py              # アプリケーションエントリーポイント
├── config.py            # 設定管理
├── core/                # ビジネスロジック
├── ui/                  # ユーザーインターフェース
├── utils/               # 共通ユーティリティ
└── services/            # アプリケーションサービス
```

### 2.3 動作環境

| 項目 | 仕様 |
|------|------|
| **実行環境** | Docker Container |
| **プログラミング言語** | Python 3.11 |

## 3. 機能詳細設計

### 3.1 動画ファイル選択機能

#### 3.1.1 機能概要

ユーザーが処理対象の動画を選択する機能です。ローカルファイルとYouTube動画の両方に対応します。

#### 3.1.2 処理仕様

| 処理名 | 説明 | 備考 |
|--------|------|------|
| **ローカルファイル選択** | videosフォルダ内の動画をドロップダウンで選択 | Docker環境 |
| **YouTube URL入力** | URLを入力して動画をダウンロード | 別タブで表示 |
| **ファイル検証** | 形式・サイズ・存在確認 | エラー時は詳細表示 |

#### 3.1.3 入出力仕様

**入力**
| 項目 | 型 | 説明 | 制約 |
|------|-----|------|------|
| ファイルパス | 文字列 | 動画ファイルの場所 | 最大255文字 |
| YouTube URL | 文字列 | YouTube動画のURL | 有効なURL形式 |

**出力**
| 項目 | 型 | 説明 |
|------|-----|------|
| 動画情報 | オブジェクト | パス、時間、サイズ、形式 |
| エラー情報 | 文字列 | エラーの詳細（エラー時のみ） |

### 3.2 文字起こし機能

#### 3.2.1 機能概要

動画から音声を抽出し、単語レベルの精密な文字起こしを行います。ローカル処理とAPI処理の2つのモードがあります。

#### 3.2.2 処理フロー

```
1. キャッシュ確認
   ├─ キャッシュあり → キャッシュから読み込み
   └─ キャッシュなし → 文字起こし処理へ

2. 音声抽出
   └─ 動画 → WAVファイル（16kHz、モノラル）

3. 文字起こし処理
   ├─ ローカルモード
   │   ├─ 無音部分で分割（最大10分）
   │   ├─ 並列処理（最大4ワーカー）
   │   └─ アライメント処理
   └─ APIモード
       ├─ 料金確認ダイアログ
       ├─ API呼び出し
       └─ アライメント処理

4. キャッシュ保存
```

#### 3.2.3 処理仕様表

| モード | 処理時間 | 精度 | コスト | 特徴 |
|--------|----------|------|--------|------|
| **ローカル** | 90分動画を15-30分 | 中～高 | 無料 | GPU利用可能 |
| **API** | 90分動画を5-10分 | 高 | $0.006/分 | 安定・高速 |

#### 3.2.4 並列処理の仕様

| 項目 | 設定値 | 説明 |
|------|--------|------|
| **最大ワーカー数** | 4 | CPU数の75%まで |
| **チャンクサイズ** | 最大10分 | 無音部分で自然に分割 |
| **メモリ制限** | Docker割当の80% | 自動調整 |

### 3.3 テキスト編集機能

#### 3.3.1 機能概要

文字起こし結果から必要な部分だけを抽出するための編集機能です。左右分割画面で直感的に操作できます。

#### 3.3.2 画面構成

```
┌─────────────────────────────────────────────────┐
│                 テキスト編集                      │
├─────────────────────┬───────────────────────────┤
│   文字起こし結果    │     編集テキスト          │
│   （読み取り専用）  │    （編集可能）           │
│                     │                           │
│  [自動スクロール]   │   [文字数: XXX]           │
└─────────────────────┴───────────────────────────┘
         [差分検出] [プレビュー] [不一致削除]
```

#### 3.3.3 差分検出アルゴリズム

1. **テキスト正規化**
   - 全角→半角変換
   - 空白の統一
   - 改行の除去

2. **マッチング処理**
   - 最長共通部分列（LCS）アルゴリズム使用
   - 文字単位での比較
   - 80%以上の一致で部分マッチ判定

3. **時間範囲の特定**
   - マッチした文字の開始・終了時間を取得
   - 前後0.1秒のバッファを追加

### 3.4 タイムライン編集機能

#### 3.4.1 機能概要

切り抜き部分の開始・終了時間をフレーム単位で微調整する機能です。

#### 3.4.2 調整方法

| 操作方法 | 調整単位 | ショートカット |
|----------|----------|----------------|
| **数値入力** | 秒単位（小数可） | - |
| **ボタン操作** | ±1, ±5, ±30フレーム | - |
| **キーボード** | ±1フレーム | ←/→ |
| **Shift+キー** | ±5フレーム | Shift+←/→ |
| **Ctrl+キー** | ±30フレーム | Ctrl+←/→ |

#### 3.4.3 波形表示仕様

| 項目 | 仕様 |
|------|------|
| **サンプリングレート** | 100Hz（0.01秒単位） |
| **表示レベル** | 0-1に正規化 |
| **色分け** | 音声部分：青、無音部分：グレー |

### 3.5 無音削除機能

#### 3.5.1 機能概要

動画から無音部分を自動的に検出し、削除する機能です。

#### 3.5.2 検出パラメータ

| パラメータ | デフォルト値 | 調整範囲 | 説明 |
|------------|-------------|----------|------|
| **閾値** | -35dB | -60～-20dB | 無音判定の音量レベル |
| **最小無音時間** | 0.3秒 | 0.1～2.0秒 | 無音と判定する最小時間 |
| **最小セグメント時間** | 0.3秒 | 0.1～5.0秒 | 残す音声の最小時間 |
| **開始パディング** | 0秒 | 0～2.0秒 | セグメント開始前の余白 |
| **終了パディング** | 0秒 | 0～2.0秒 | セグメント終了後の余白 |

#### 3.5.3 処理結果

削除前後の比較：

| 項目 | 削除前 | 削除後（例） |
|------|--------|--------------|
| **総時間** | 90分 | 60分 |
| **セグメント数** | 1 | 150 |
| **削除率** | - | 33% |

### 3.6 エクスポート機能

#### 3.6.1 出力形式一覧

| 形式 | 拡張子 | 用途 | 特徴 |
|------|--------|------|------|
| **動画ファイル** | .mp4 | 最終成果物 | H.264/AAC、高品質 |
| **FCPXML** | .fcpxml | DaVinci Resolve/Final Cut Pro | v1.9形式 |
| **Premiere XML** | .xml | Adobe Premiere Pro | XMEML形式 |
| **字幕** | .srt | 汎用字幕 | UTF-8、改行対応 |

#### 3.6.2 エクスポートオプション

| オプション | 説明 | デフォルト |
|------------|------|------------|
| **動画品質** | 無劣化コピー or 再エンコード | 無劣化 |
| **無音削除** | 適用する/しない | しない |
| **字幕埋め込み** | XMLに字幕トラック追加 | しない |

## 5. エラー処理設計

### 5.1 エラー分類と対処

| エラー種別 | 発生条件 | 対処方法 | ユーザー通知 |
|------------|----------|----------|--------------|
| **入力検証エラー** | 無効なファイル形式 | 処理中止 | エラー詳細を表示 |
| **メモリ不足** | 大容量ファイル処理時 | チャンクサイズ縮小 | 設定変更を提案 |
| **API接続エラー** | ネットワーク障害 | 3回リトライ→ローカル | フォールバック通知 |
| **FFmpegエラー** | コーデック非対応 | 代替形式で再試行 | 技術詳細を表示 |

### 5.2 リカバリー機能

| 機能 | 説明 | 保存期間 |
|------|------|----------|
| **チェックポイント** | 各処理段階で状態保存 | 24時間 |
| **部分キャッシュ** | 処理済み部分を保存 | 無期限 |
| **セッション復元** | ブラウザリロード対応 | セッション中 |
