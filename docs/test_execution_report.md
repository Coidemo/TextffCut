## 実装とテストの主な差異

### 1. Duration値オブジェクト
- **タイムコード形式**: 実装は`HH:MM:SS:FF`（フレーム）、テストは`HH:MM:SS.mmm`（ミリ秒）を期待
- **人間可読形式**: 実装は`0ms`、`1.5s`など、テストは`0.0秒`、`1.5秒`を期待
- **エラーハンドリング**: 実装は負の値をクランプ、テストは例外を期待

### 2. TimeRange値オブジェクト
- 多くのメソッドが未実装または異なる実装
- 比較演算子が未実装

### 3. VideoSegmentエンティティ
- `text`属性が実装に存在しない
- クラスメソッドのシグネチャが異なる

### 4. TranscriptionResultエンティティ
- コンストラクタが`duration`パラメータを受け付けない（プロパティとして計算される）
