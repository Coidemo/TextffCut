# TextffCut クイックスタートガイド

5分でTextffCutを使い始めるためのガイドです。

## 🚀 3ステップで始める

### ステップ1: インストール（初回のみ）

```bash
# リポジトリをダウンロード
git clone https://github.com/Coidemo/TextffCut.git
cd TextffCut

# 自動インストール
./scripts/install.sh
```

### ステップ2: 起動

```bash
# アプリを起動
./scripts/start.sh
```

ブラウザが開き、TextffCutが表示されます。

### ステップ3: 動画を処理

1. **動画パスを入力**
2. **「文字起こしを開始」をクリック**
3. **必要な部分をコピー＆ペースト**
4. **「処理を開始」をクリック**

完了！切り抜き動画が作成されます。

## 🎯 基本的な使い方

### 動画の切り抜き（最も簡単）

```
1. 動画をアップロード
2. 文字起こし（自動）
3. 残したい部分を選択
4. 実行
```

### 無音も削除したい場合

サイドバーで「切り抜いた動画から無音部分を削除」をチェック

## 💡 便利な機能

### 複数箇所を切り抜き

残したい部分を複数選択してペースト：
```
最初の部分...
2番目の部分...
3番目の部分...
```

### セクション分割

`---`で区切ると個別ファイルに：
```
セクション1
---
セクション2
---
セクション3
```

### APIで高速化

サイドバーで「APIモードを使用」をチェック（要OpenAI APIキー）

## 📁 出力場所

処理後のファイルは動画と同じフォルダに作成されます：
```
動画ファイル.mp4
動画ファイル_TextffCut/
  ├── 動画ファイル_cut.mp4      ← 切り抜き結果
  ├── 動画ファイル_no_silence.mp4 ← 無音削除版
  └── 動画ファイル.fcpxml        ← 編集ソフト用
```

## ❓ よくある質問

**Q: どんな動画形式に対応？**
A: MP4, MOV, AVI, MKV, WMV

**Q: 処理時間は？**
A: 30分動画で約5-10分（モデルとPCスペックによる）

**Q: 無料？**
A: ローカル版は完全無料。APIモードは従量課金（$0.006/分）

**Q: 文字起こしの精度は？**
A: OpenAI Whisperベース。日本語対応で高精度

## 🔧 推奨設定

初心者向け：
- **Whisperモデル**: medium
- **無音閾値**: -35dB（デフォルト）
- **PAD**: 0.1秒（デフォルト）
