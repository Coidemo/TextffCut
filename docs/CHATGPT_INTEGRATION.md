# ChatGPT連携機能

## 概要

TextffCutのChatGPT連携機能により、文字起こし結果や切り抜き箇所から、AIを活用してバズる切り抜き案やタイトルを生成できます。

## 機能

### 1. 外部タブモード（無料）

ChatGPTのWebサイトを新規タブで開き、プロンプトをコピーして利用する基本的な連携方法です。

**使い方：**
1. 文字起こしを実行
2. 切り抜き箇所を指定（オプション）
3. 右カラム下部の「ChatGPT連携」セクションで「外部タブで開く（無料）」を選択
4. 使用するテキスト（文字起こし全文 or 切り抜き箇所）を選択
5. プロンプトテンプレートを選択
6. 「プロンプトをコピー」ボタンをクリック
7. 「ChatGPTを開く」ボタンで新規タブを開く
8. ChatGPTにプロンプトをペースト

**利用可能なプロンプトテンプレート：**
- バズる切り抜き案
- タイトル提案
- 要約
- カスタムプロンプト

### 2. API統合モード（有料）

OpenAI APIを使用して、TextffCut内で直接結果を取得できる高度な連携方法です。

**特徴：**
- 元の文字起こしから単語を削除するだけでバズる切り抜きを生成
- 文字の追加や改変は行わず、削除のみで文章を構築
- 複数の提案を順次表示可能
- ワンクリックで切り抜き箇所に設定可能

**使い方：**
1. OpenAI APIキーを設定（初回のみ）
2. 「API統合（有料）」を選択
3. 使用するテキストを選択
4. 機能を選択：
   - **バズる切り抜き案を生成**: 元の文字から単語削除のみで切り抜き案を作成
   - **タイトル案を生成**: 切り抜きテキストに対するタイトルを提案
5. 「生成」ボタンをクリック

**バズる切り抜き案の特徴：**
- 元の文字起こしにある単語のみを使用
- 単語の順序は元のテキストと同じ
- 30秒〜1分程度で話せる長さに調整
- 「前の案」「次の案」ボタンで複数案を確認
- 「この案を切り抜き箇所に設定」で即座に適用

## 料金

### 外部タブモード
- 無料（ChatGPTの無料プランで利用可能）

### API統合モード
- GPT-4o-mini使用: 約$0.001-0.005/リクエスト
- 日本円換算: 約0.15円〜0.75円/リクエスト（$1=150円として）

## APIキーの取得方法

1. [OpenAI Platform](https://platform.openai.com/)にアクセス
2. アカウントを作成またはログイン
3. 左メニューの「API keys」をクリック
4. 「Create new secret key」をクリック
5. 生成されたキー（sk-で始まる文字列）をコピー

## 技術的な仕組み

### 単語削除アルゴリズム

API統合モードの「バズる切り抜き案」機能では、以下のアルゴリズムを使用：

1. **セグメント分割**: 元のテキストを文節・単語単位でセグメントに分割
2. **インデックス管理**: 各セグメントに0から始まるインデックスを付与
3. **ChatGPT処理**: 使用するセグメントのインデックスリストを生成
4. **テキスト再構築**: 選択されたセグメントのみを元の順序で結合

これにより、元の文字起こしにない言葉は一切使わず、削除のみで新しい切り抜きを作成できます。

## トラブルシューティング

### プロンプトのコピーができない
- ブラウザのセキュリティ設定を確認
- 手動でテキストを選択してコピー

### API統合でエラーが発生する
- APIキーが正しく設定されているか確認
- OpenAIアカウントに十分なクレジットがあるか確認
- ネットワーク接続を確認

### 生成される切り抜きが不自然
- より長い元テキストを使用
- 異なるセクションから再生成を試す
