# SRT字幕ファイル生成機能

## 概要
TextffCutに字幕ファイル（SRT形式）のエクスポート機能を追加しました。文字起こし結果のタイムスタンプを活用して、高品質な字幕ファイルを素早く生成できます。

## 機能の特徴
- **高速生成**: 既存の文字起こしタイムスタンプを活用
- **切り抜き対応**: 指定した時間範囲のみの字幕を生成
- **標準形式**: 汎用的なSRT（SubRip）形式で出力

## 使い方
1. 動画の文字起こしを実行
2. 切り抜きたい部分のテキストを編集エリアに入力
3. 出力形式で「SRTファイル」を選択
4. 「処理を実行」をクリック

## 出力例
```srt
1
00:00:00,000 --> 00:00:02,500
こんにちは、今日は良い天気ですね。

2
00:00:03,000 --> 00:00:05,800
明日も晴れるといいですね。

3
00:00:06,200 --> 00:00:09,100
週末は公園に行きたいです。
```

## 技術仕様
- **ファイル形式**: SRT (SubRip Text)
- **文字コード**: UTF-8
- **タイムコード形式**: HH:MM:SS,mmm
- **対応言語**: 全言語（WhisperX/OpenAI APIがサポートする言語）

## 活用例
- 動画編集ソフトでの字幕追加
- YouTube字幕のアップロード
- 多言語字幕の作成ベース
- アクセシビリティ向上

## 注意事項
- 文字起こしの精度に依存します
- 無音削除を適用した場合、字幕のタイミングも調整されます
- 長い文章は適切に分割されて表示されます
