# SRT字幕ファイルエクスポート機能

## 概要
TextffCutに字幕ファイル（SRT形式）のエクスポート機能を追加しました。文字起こし結果から直接SRTファイルを生成でき、行数や1行あたりの文字数を柔軟に設定できます。

## 機能の特徴

### 1. 柔軟な表示設定
- **1字幕の最大行数**: 1〜4行で設定可能
- **1行の最大文字数**: 10〜80文字で設定可能
- デフォルト値: 2行、40文字/行

### 2. 賢いテキスト分割
#### 日本語の場合
- 句読点（。、！？）を優先的に改行位置として使用
- 読点（、）も考慮して自然な位置で分割
- 文字数制限に達した場合は適切な位置で強制分割

#### 英語等の場合
- 単語単位で分割（単語の途中では切らない）
- スペースを考慮した自然な改行

### 3. 時間範囲フィルタリング対応
- 切り抜き部分のみの字幕生成が可能
- 複数の時間範囲を指定した場合も対応

## 使い方

### UIでの操作
1. 文字起こし完了後、「出力形式」で「SRTファイル」を選択
2. 右側に表示される字幕設定で行数と文字数を調整
3. 「処理を実行」ボタンをクリック

### 出力例

#### 設定: 2行、40文字
```srt
1
00:00:00,000 --> 00:00:03,500
こんにちは、今日は天気がいいですね。
散歩に行きましょうか。

2
00:00:03,500 --> 00:00:08,200
This is a test of the subtitle
generation system with English text

3
00:00:08,200 --> 00:00:15,000
長い日本語のテキストです。これは字幕生成システムの
テストで、設定した文字数と行数に応じて適切に...
```

#### 設定: 1行、20文字
```srt
1
00:00:00,000 --> 00:00:03,500
こんにちは、今日は天気がいい...

2
00:00:03,500 --> 00:00:08,200
This is a test of...

3
00:00:08,200 --> 00:00:15,000
長い日本語のテキストです。...
```

## 技術仕様

### SRTフォーマット
```
字幕番号
開始時刻 --> 終了時刻
字幕テキスト（複数行可）
[空行]
```

### タイムスタンプ形式
- `HH:MM:SS,mmm` （時:分:秒,ミリ秒）
- 例: `00:01:23,456`

### 実装詳細
- **SRTExporter**クラス: `core/export.py`
  - テキスト分割ロジック
  - 日本語判定
  - SRTフォーマット生成
  
- **ExportService**統合: `services/export_service.py`
  - 他のエクスポート形式との統一的なインターフェース
  - 検証機能

- **UI統合**: `ui/components.py`, `main.py`
  - 設定UIの追加
  - ワークフローへの統合
