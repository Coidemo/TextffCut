# TextffCut プロジェクト概要

## プロジェクトの目的
**TextffCut** は動画の文字起こしと切り抜きを効率化するツールです。テキスト差分による動画切り抜き（Text + diff + Cut）を実現します。

### 主な機能
- 90分程度の長時間動画から必要な部分だけを抽出
- 無音部分を自動削除してタイトな編集素材を作成  
- DaVinci ResolveやFinal Cut Pro用のFCPXML生成
- AIバズクリップ生成（GPT-4oによる面白い切り抜き候補の自動提案）

## 現在のバージョン
- **v1.1.0-dev** (開発版)
- **v0.9.6** (最新安定版)

## 動作モード
1. **APIモード**: OpenAI Whisper API（高速・安定）
2. **ローカルモード**: WhisperX（無料・プライベート）

## 処理速度
- APIモード: 90分動画を約2-5分で処理（約81円）
- GPU（NVIDIA RTX 40系）: 約5-10分
- GPU（M1/M2 Mac）: 約10-15分  
- CPU: 約30-120分（性能により大幅に変動）
